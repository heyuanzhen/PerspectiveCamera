# PerspectiveCamera



## Usage 
- Install Requirements

Python Version: 3.6
```commandline
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

- Prepare Raw Data

Put all files that need to be transformed into `./resources/raw/`

- Run the Script
```commandline
python main.py
```

- Get Processed Files

The transformed data are put into `./resources/processed/`
